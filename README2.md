# Pattern Generator and Number Analyzer

## Overview

This Python program allows users to: 1. Generate a star (\*) pattern. 2.
Analyze a range of numbers (even/odd + sum). 3. Exit the program.

------------------------------------------------------------------------

## Features

### 1. Pattern Generator

-   Takes number of rows as input.
-   Prints a triangle star pattern.
-   Validates input (must be positive).

Example: * ** *** \*\*\*\*

------------------------------------------------------------------------

### 2. Number Analyzer

-   Takes starting and ending number.
-   Skips negative numbers using `continue`.
-   Ignores zero using `pass`.
-   Identifies even and odd numbers.
-   Calculates total sum of valid numbers.

------------------------------------------------------------------------

### 3. Exit Option

-   Ends the program safely.

------------------------------------------------------------------------

## Concepts Used

-   Loops (`while`, `for`)
-   Conditional statements (`if`, `elif`, `else`)
-   `break`, `continue`, `pass`
-   Input/Output handling
-   String formatting (f-strings)

------------------------------------------------------------------------

## How to Run

1.  Install Python.

2.  Copy the code into a `.py` file.

3.  Run the file:

        python filename.py
